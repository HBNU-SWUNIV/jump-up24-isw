## Project Background

![image](https://github.com/user-attachments/assets/4fafde26-be74-4bd2-a2a0-311558764e65)


본 연구는 데이터 부족 문제를 해결하고 CNN 모델의 객체 탐지 성능을 개선하기 위해 진행되었습니다.  
자율주행 자동차와 같은 이동 로봇에서 신호등과 같은 객체를 정확히 탐지하는 것은 중요한 과제입니다. 특히, EV3 로봇이 신호등을 구분하는 성능을 개선하기 위해, 가우시안 블러 전처리 기법을 활용하여 데이터를 증강하고 모델의 효율성을 향상시키는 방법을 제안했습니다.

- 신호등 객체를 제외한 배경에 블러링을 적용해 데이터를 증강.

#### **3. 실험 결과 분석**  
- 가우시안 블러 적용 범위(사진 전체 vs. 객체 제외 배경)에 따른 성능 차이를 정량적으로 비교.

### 연구 기여  

이 연구는 데이터 전처리의 효과와 CNN 모델 성능 향상을 동시에 확인할 수 있는 중요한 기여를 했습니다.  
이를 통해 적은 데이터로도 높은 정확도를 유지하며, 객체 탐지의 효율성을 극대화할 수 있는 가능성을 입증했습니다.


---

## System Model  
CNN 모델 학습과 테스트는 EDGE IMPULSE 플랫폼에서 진행되었으며, 전처리 기법에 따른 성능 변화를 분석하였습니다.

### 1. CNN 모델 구조  
- **기본 구조**: EDGE IMPULSE에서 제공되는 CNN 모델 사용.  
- **입력 데이터 크기**: 48x48로 통일하여 계산 속도와 일관성을 확보.  
- **학습 파라미터**:  
  - Epoch: 10  
  - Learning Rate: 0.0005  

### 2. 데이터 전처리 과정  
1. **사진 전체 블러링**  
   - 이미지 전체에 가우시안 블러 적용.  
   - 객체와 배경을 모두 흐리게 처리.  

2. **객체 제외 배경 블러링**  
   - **HSV 변환 및 색상 마스킹**: HSV 색상 공간을 활용해 신호등 객체 탐지.  
   - **배경 블러링**: 신호등 객체를 제외한 배경에만 가우시안 블러 적용.  

### 3. 학습 및 테스트 과정  
- **데이터 업로드**: EDGE IMPULSE에서 학습 및 테스트 데이터를 각각 업로드하고 라벨 설정.  
- **Impulse 생성**: 입력 블록에서 `IMAGES`를 선택, 입력 크기를 48x48로 설정.  
- **모델 학습**: 전처리된 데이터를 학습 데이터로 사용하여 CNN 모델 훈련.  
- **모델 테스트**: 테스트 데이터를 통해 정확도(Accuracy)와 손실(Loss)를 검증.  

## Numerical Results

본 연구에서는 CNN 모델의 학습 및 테스트를 위해 전처리 방식별로 데이터를 나누어 성능을 비교했습니다. 성능은 정확도(Accuracy)와 손실(Loss)을 기준으로 분석했습니다.

---

### 1. 데이터셋 구성

- **원본 데이터**  
  - EV3 로봇 카메라로 촬영된 신호등 원본 이미지 1100장 사용.

- **사진 전체 블러링 데이터**  
  - 1000장의 이미지를 대상으로 사진 전체에 가우시안 블러를 적용한 데이터셋 생성.

- **객체 제외 배경 블러링 데이터**  
  - 1000장의 이미지를 대상으로 객체를 제외한 배경만 가우시안 블러를 적용한 데이터셋 생성.

---

### 2. 객체 탐지 성공률

- **평가 기준**  
  - Intersection over Union (IoU) ≥ 0.5로 객체 탐지 성공 여부를 평가.

- **결과**  
  - **빨간 신호등**: 1000장의 객체 제외 배경 블러링 데이터 중 846장이 탐지 성공.  
  - **초록 신호등**: 1000장의 객체 제외 배경 블러링 데이터 중 888장이 탐지 성공.  

---

### 3. 학습 데이터와 테스트 데이터 결과 비교

CNN 모델 학습과 테스트는 Epoch 10~25 구간에서 정확도와 손실 값을 기준으로 비교했습니다.

- **원본 데이터 (1100장)**  
  - 정확도: Epoch=25 기준 **86.53%**  
  - 손실: Epoch=25 기준 **0.19**  
  - 가장 안정적인 성능을 보였으나, 노이즈 제거가 부족해 다른 전처리 데이터보다 손실 값이 상대적으로 높음.

- **사진 전체 블러링 데이터 (1000장)**  
  - 정확도: Epoch=25 기준 **85.05%**  
  - 손실: Epoch=25 기준 **0.15**  
  - 객체 정보와 배경 정보를 함께 블러 처리하면서 주요 정보 손실이 발생해 성능이 감소.

- **객체 제외 배경 블러링 데이터 (1000장)**  
  - 정확도: Epoch=25 기준 **87.37%**  
  - 손실: Epoch=25 기준 **0.09**  
  - 배경 노이즈를 효과적으로 제거하면서 객체 정보를 보존해 가장 높은 정확도와 가장 낮은 손실 값을 기록.

---

### 4. Epoch별 성능 비교

Epoch 10, 15, 20, 25 구간에서 10회 반복 학습을 통해 정확도와 손실 값의 평균을 비교했습니다.

- **정확도 평균 및 표준편차**  
  - 원본 데이터: 평균 정확도 **86.533%** (표준편차 ±1.465).  
  - 사진 전체 블러링 데이터: 평균 정확도 **85.049%** (표준편차 ±2.143).  
  - 객체 제외 배경 블러링 데이터: 평균 정확도 **86.733%** (표준편차 ±1.993).  

- **손실 값 비교**  
  - 객체 제외 배경 블러링 데이터가 가장 낮은 손실 값 (**0.09**)을 기록, 데이터 품질 개선의 효과를 입증.

---

### 5. 데이터 감소 영향 분석

- 객체 제외 배경 블러링 데이터는 원본 데이터 대비 약 10% 적은 데이터로 학습했음에도 동등 이상의 정확도를 유지.  
- 데이터의 효율적인 활용 가능성을 입증하며, 데이터 부족 상황에서도 높은 성능을 발휘할 수 있음을 확인.

---

### 6. 결과 요약

- 객체 제외 배경 블러링 기법은 CNN 모델이 배경 노이즈를 줄이고, 객체 정보를 효과적으로 학습하도록 도움.  
- 전체 블러링 데이터는 일부 성능 저하를 초래했지만, 데이터 품질 관리에 유용할 가능성이 있음.  
- Epoch=25 기준으로 객체 제외 배경 블러링 데이터가 가장 높은 정확도와 가장 낮은 손실 값을 기록, 제안 기법의 효과를 입증함.


---

## Conclusion  
본 연구는 객체 제외 배경 블러링 기법이 CNN 모델의 성능 향상에 효과적임을 입증하였습니다. 데이터 부족 상황에서도 안정적인 정확도를 유지하며, 가우시안 블러 전처리 기법은 자율주행 및 로봇 비전 시스템에서 활용 가능성이 높습니다.

---

### 1. 가우시안 블러 전처리 기법의 성능 입증

- **객체 제외 배경 블러링 기법**은 CNN 모델이 배경 노이즈를 줄이고 객체 정보를 효과적으로 학습할 수 있도록 지원.  
  - Epoch=25 기준, 정확도 **87.37%**와 손실 값 **0.09**를 기록하며 가장 높은 성능을 보임.  
  - 데이터 양이 약 10% 감소했음에도 원본 데이터 대비 동등 이상의 정확도를 유지.  

- **사진 전체 블러링 기법**은 성능 저하를 초래했지만, 배경 정보를 흐리게 하는 데이터 품질 개선의 가능성을 확인.  

---

### 2. 데이터 전처리를 통한 효율성 개선

- 적은 데이터로도 높은 정확도를 달성할 수 있는 데이터 증강 방법론을 제시.  
- **객체 제외 배경 블러링 기법**은 적은 데이터셋에서도 원본 데이터 대비 **0.84% 높은 정확도**를 달성.  
- 데이터 편향 문제를 줄이고, 모델의 안정성을 높이는 데 기여.  
- 다양한 전처리 방법의 비교를 통해 CNN 모델 학습에서 데이터 품질 관리의 중요성을 확인.  

---

### 3. 자율주행 및 로봇 비전 응용 가능성

- 본 연구에서 제안된 기법은 자율주행 자동차와 로봇 비전 시스템에 바로 적용 가능.  
  - 예: 자율주행 자동차에서 신호등 탐지의 정확도를 높이고, 노이즈에 민감한 환경에서 안정적으로 작동.  
- 특정 객체를 강조하고 배경을 줄이는 방식은 다양한 객체 탐지 응용 분야에서도 활용 가능.  
  - 예: 교통 표지판 탐지, 공장 자동화 시스템에서의 객체 분류 등.
