## Project Background  
본 연구는 데이터 부족 문제를 해결하고 CNN 모델의 객체 탐지 성능을 개선하기 위해 가우시안 블러 전처리 기법을 제안하였습니다. 자율주행 자동차와 같은 이동 로봇에서 신호등을 탐지하는 작업은 중요한 과제이며, 기존 HSV 변환 방식의 한계를 극복하고 효율적인 데이터 증강 기법을 연구하였습니다.

### 연구 배경  
1. **HSV 전처리 한계**  
   HSV 변환은 색상을 조정하는 데 효과적이지만, 데이터 노이즈를 증가시키고 비현실적인 결과를 초래하여 CNN 모델 성능을 약 38.73%까지 감소시켰습니다.

2. **데이터 증강 필요성**  
   EV3 로봇을 위한 신호등 이미지 데이터는 한정되어 있으며, 데이터 증강을 통해 학습 결과를 개선하고 모델의 성능을 높이고자 했습니다.

3. **가우시안 블러 기법 도입**  
   고주파 성분 제거 및 데이터 품질 개선을 위해 가우시안 블러를 활용하여, 사진 전체 블러링과 객체 제외 배경 블러링 두 가지 방식을 실험했습니다.

---

## System Model  
CNN 모델 학습과 테스트는 EDGE IMPULSE 플랫폼에서 진행되었으며, 전처리 기법에 따른 성능 변화를 분석하였습니다.

### 1. CNN 모델 구조  
- **기본 구조**: EDGE IMPULSE에서 제공되는 CNN 모델 사용.  
- **입력 데이터 크기**: 48x48로 통일하여 계산 속도와 일관성을 확보.  
- **학습 파라미터**:  
  - Epoch: 10  
  - Learning Rate: 0.0005  

### 2. 데이터 전처리 과정  
1. **사진 전체 블러링**  
   - 이미지 전체에 가우시안 블러 적용.  
   - 객체와 배경을 모두 흐리게 처리.  

2. **객체 제외 배경 블러링**  
   - **HSV 변환 및 색상 마스킹**: HSV 색상 공간을 활용해 신호등 객체 탐지.  
   - **배경 블러링**: 신호등 객체를 제외한 배경에만 가우시안 블러 적용.  

전처리된 데이터는 아래 폴더 구조로 저장:  
- `blurred_image_full_red/`: 빨간 신호등 전체 블러링.  
- `blurred_image_full_green/`: 초록 신호등 전체 블러링.  
- `blurred_image_bg_red/`: 빨간 신호등 객체 제외 배경 블러링.  
- `blurred_image_bg_green/`: 초록 신호등 객체 제외 배경 블러링.  

### 3. 학습 및 테스트 과정  
- **데이터 업로드**: EDGE IMPULSE에서 학습 및 테스트 데이터를 각각 업로드하고 라벨 설정.  
- **Impulse 생성**: 입력 블록에서 `IMAGES`를 선택, 입력 크기를 48x48로 설정.  
- **모델 학습**: 전처리된 데이터를 학습 데이터로 사용하여 CNN 모델 훈련.  
- **모델 테스트**: 테스트 데이터를 통해 정확도(Accuracy)와 손실(Loss)를 검증.  

---## Numerical Results

본 연구에서는 CNN 모델의 학습 및 테스트를 위해 전처리 방식별로 데이터를 나누어 성능을 비교했습니다. 성능은 정확도(Accuracy)와 손실(Loss)을 기준으로 분석했습니다.

---

### 1. 데이터셋 구성

- **원본 데이터**  
  - EV3 로봇 카메라로 촬영된 신호등 원본 이미지 1100장 사용.

- **사진 전체 블러링 데이터**  
  - 1000장의 이미지를 대상으로 사진 전체에 가우시안 블러를 적용한 데이터셋 생성.

- **객체 제외 배경 블러링 데이터**  
  - 1000장의 이미지를 대상으로 객체를 제외한 배경만 가우시안 블러를 적용한 데이터셋 생성.

---

### 2. 객체 탐지 성공률

- **평가 기준**  
  - Intersection over Union (IoU) ≥ 0.5로 객체 탐지 성공 여부를 평가.

- **결과**  
  - **빨간 신호등**: 1000장의 객체 제외 배경 블러링 데이터 중 846장이 탐지 성공.  
  - **초록 신호등**: 1000장의 객체 제외 배경 블러링 데이터 중 888장이 탐지 성공.  

---

### 3. 학습 데이터와 테스트 데이터 결과 비교

CNN 모델 학습과 테스트는 Epoch 10~25 구간에서 정확도와 손실 값을 기준으로 비교했습니다.

- **원본 데이터 (1100장)**  
  - 정확도: Epoch=25 기준 **86.53%**  
  - 손실: Epoch=25 기준 **0.19**  
  - 가장 안정적인 성능을 보였으나, 노이즈 제거가 부족해 다른 전처리 데이터보다 손실 값이 상대적으로 높음.

- **사진 전체 블러링 데이터 (1000장)**  
  - 정확도: Epoch=25 기준 **85.05%**  
  - 손실: Epoch=25 기준 **0.15**  
  - 객체 정보와 배경 정보를 함께 블러 처리하면서 주요 정보 손실이 발생해 성능이 감소.

- **객체 제외 배경 블러링 데이터 (1000장)**  
  - 정확도: Epoch=25 기준 **87.37%**  
  - 손실: Epoch=25 기준 **0.09**  
  - 배경 노이즈를 효과적으로 제거하면서 객체 정보를 보존해 가장 높은 정확도와 가장 낮은 손실 값을 기록.

---

### 4. Epoch별 성능 비교

Epoch 10, 15, 20, 25 구간에서 10회 반복 학습을 통해 정확도와 손실 값의 평균을 비교했습니다.

- **정확도 평균 및 표준편차**  
  - 원본 데이터: 평균 정확도 **86.533%** (표준편차 ±1.465).  
  - 사진 전체 블러링 데이터: 평균 정확도 **85.049%** (표준편차 ±2.143).  
  - 객체 제외 배경 블러링 데이터: 평균 정확도 **86.733%** (표준편차 ±1.993).  

- **손실 값 비교**  
  - 객체 제외 배경 블러링 데이터가 가장 낮은 손실 값 (**0.09**)을 기록, 데이터 품질 개선의 효과를 입증.

---

### 5. 데이터 감소 영향 분석

- 객체 제외 배경 블러링 데이터는 원본 데이터 대비 약 10% 적은 데이터로 학습했음에도 동등 이상의 정확도를 유지.  
- 데이터의 효율적인 활용 가능성을 입증하며, 데이터 부족 상황에서도 높은 성능을 발휘할 수 있음을 확인.

---

### 6. 결과 요약

- 객체 제외 배경 블러링 기법은 CNN 모델이 배경 노이즈를 줄이고, 객체 정보를 효과적으로 학습하도록 도움.  
- 전체 블러링 데이터는 일부 성능 저하를 초래했지만, 데이터 품질 관리에 유용할 가능성이 있음.  
- Epoch=25 기준으로 객체 제외 배경 블러링 데이터가 가장 높은 정확도와 가장 낮은 손실 값을 기록, 제안 기법의 효과를 입증함.
